🎮 How to Play

🖥 Desktop Keyboard Controls

↑ ↓ ← →Move / Navigate
EnterConfirm / Select
BackspaceGo back / Cancel
1 – 9Numeric keypad
0 / * / #Special keys
L keyLeft soft key
R keyRight soft key
MouseTap / click (touch games)

✋ L & R Soft Key Buttons

L buttonLeft menu / options key
R buttonRight back / action key
Drag L / RMove button to any position
L / R keysKeyboard shortcut equivalent

💡 Tip: The L and R buttons on the left and right edges of the screen are the soft keys used in classic J2ME games. Drag them up or down to position them wherever is most comfortable. On touch-screen games, use your mouse to click or drag directly on the game canvas.
